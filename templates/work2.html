002 : SMS USING PYTHON

This project sends an sms using python

This project is built using python language and it's libraries such as smtplib, EmailMessage, Template

Scroll below for screenshots


Import Twilio Library:

Import the Client class from the Twilio REST library.

Twilio Account Credentials:

Set your Twilio account SID and authentication token as variables (account_SID and auth_token).

Create Twilio Client:

Create a Twilio client instance using the provided account SID and authentication token.

Compose and Send SMS Message:

Use the Twilio client to create an SMS message by calling the create method on the messages attribute. Set the message body using the body parameter. Specify the Twilio phone number from which the SMS will be sent (from_). Provide the recipient's phone number (to). The create method sends the SMS message, and the resulting message object is stored in the message variable.

Print Message SID:

Print the SID (Service Identifier) of the sent message using print(message.sid). The SID uniquely identifies the created SMS message.

Ensure that your Twilio credentials are kept secure, and make sure that you have a valid and active Twilio account with the necessary permissions to send SMS messages. The printed message SID can be useful for tracking the status and details of the sent message.
