004 : SCRAPE

This project is used to scrape web page

This project is built using python language and it's library known as BeautifulSoup

Scroll below for screenshots


Import Libraries:

The script begins by importing the necessary Python libraries: requests for making HTTP requests and BeautifulSoup for HTML parsing, pprint is imported to pretty-print the output.

HTTP Requests:

Two HTTP requests (res and res2) are made to the Hacker News website to fetch the HTML content of the first and second pages.

HTML Parsing:

The HTML content of each page is parsed using BeautifulSoup, and the resulting soup objects are named soup and soup2.

Selecting Elements:

CSS selectors are used to extract relevant information from the HTML, specifically article links and subtext. These elements are stored in lists named links, subtext, links2, and subtext2.

Combining Lists:

The lists from both pages are combined into mega_links and mega_subtext.

Sorting and Filtering:

A function sort_stories_by_votes is defined to sort a list of articles based on their votes in descending order. Another function create_custom_hn is defined to filter articles with votes greater than 99 and then sort them using the previous function.

Printing Results:

The sorted and filtered list of articles is printed using pprint.pprint. The output includes article titles, links, and votes, but you can modify the print statement to display only the points if desired.

Sequential Points Printing:

The script is modified to iterate through the sorted list and print only the 'votes' attribute for each article, giving you the points in sequential order.In summary, the script fetches Hacker News articles, filters them based on votes, and outputs a sorted list containing article titles, links, and votes. The modified version allows you to print only the points in sequential order.
